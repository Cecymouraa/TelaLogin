# Tela de Login - Projeto Pessoal

Este repositório contém um projeto simples de **tela de login**, criado com o objetivo de praticar a estruturação de formulários e o design de interfaces utilizando apenas **HTML** e **CSS**. 

A proposta foi desenvolver uma interface limpa, funcional e esteticamente agradável, que possa ser usada como base em projetos futuros que exijam autenticação de usuário.

Todo o design foi feito apenas com **HTML5** e **CSS3**, sem uso de JavaScript ou bibliotecas externas. 

## Tecnologias Utilizadas

- **HTML5**: estrutura da página e elementos do formulário
- **CSS3**: estilização do layout, botões, fontes, espaçamentos e responsividade
